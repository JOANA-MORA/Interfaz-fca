# SGRI — Sistema de Gestión de Recursos Informáticos
## Facultad de Contaduría y Administración — UAEMéx

Prototipo de interfaces del sistema web para la gestión de recursos informáticos de los laboratorios de cómputo de la FCA.

---

## Pantallas incluidas

| Archivo | Descripción | Rol |
|---------|-------------|-----|
| `login.html` | Inicio de sesión | Todos |
| `dashboard.html` | Panel principal con estadísticas | Administrador |
| `reportar-falla.html` | Formulario de reporte de incidencias | Alumno / Docente |
| `mantenimientos.html` | Gestión de mantenimientos | Técnico |

## Cómo ejecutar localmente

1. Clona el repositorio
2. Abre `login.html` en tu navegador
3. Usa estos correos para navegar entre vistas:
   - `admin@uaemex.mx` → Dashboard
   - `alumno@uaemex.mx` → Reportar falla
   - `tecnico@uaemex.mx` → Mantenimientos
   - `docente@uaemex.mx` → Reportar falla
   - Contraseña: cualquier texto de 4+ caracteres

## Tecnologías utilizadas

- HTML5
- CSS3 (variables, grid, flexbox)
- JavaScript vanilla
- Fuentes: DM Sans (Google Fonts)
